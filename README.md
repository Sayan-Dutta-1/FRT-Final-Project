# FRT: Microsoft Future Ready Talent Final Project by Sayan Dutta
I have created a food blog website named Hot & Spicy using HTML, CSS, JavaScript, Bootstrap and jQuery. It helps the people to feast on quality and affordable food. My website contains three pages Home, About, Contact. It contains various recipes curated by top well known chefs. It also indulges in several lifehacks of healthy eating. Following this website one can maintain a healthy scrumptious diet on a budget and without compromising on the joy of eating.
